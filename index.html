Tentang Saya

HN

Salam kenal! Saya Harits Nur Adnan, seorang siswa kelas 11 dari MAN 2 Kota Madiun yang berusia 17 tahun. Sejak menemukan dunia coding, saya jatuh cinta pada proses memecahkan masalah dan menciptakan sesuatu yang baru melalui baris-baris kode.

Bagi saya, coding bukan hanya sekadar pelajaran, tetapi juga sebuah hobi yang mengasah logika dan kreativitas. Saya sangat antusias untuk terus belajar dan mengembangkan kemampuan di bidang teknologi, serta bercita-cita untuk dapat memberikan kontribusi nyata melalui inovasi digital di masa depan.

Bahasa & Teknologi yang Saya Pelajari

Python

Dasar-dasar pemrograman, struktur data, dan logika. Mampu membuat skrip sederhana untuk otomatisasi dan analisis data dasar.

HTML

Memahami struktur fundamental halaman web. Mampu membuat kerangka website yang semantik dan terstruktur dengan baik.

JSON

Memahami format pertukaran data yang ringan. Mampu mengelola dan menstrukturkan data untuk aplikasi web.
